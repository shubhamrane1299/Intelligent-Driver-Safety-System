# Intelligent-Driver-Safety-System


AI-based Driver Monitoring System using:

- Face Recognition
- Drowsiness Detection
- Yawning Detection
- Head Pose Detection
- Mobile Usage Detection
- Voice Alerts
- Trip Logging
- Analytics Dashboard
- PDF Reports

## Technologies

- Python
- OpenCV
- Mediapipe
- YOLOv8
- SQLite
- Tkinter
- Matplotlib
